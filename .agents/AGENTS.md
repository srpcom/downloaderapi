# Project Rules - G-Drive Downloader API

- **Deployment Otomatis**: Setiap kali melakukan perubahan kode (edit berkas, penambahan fitur, perbaikan bug), agen wajib langsung menjalankan `clasp push` untuk melakukan deployment ke Google Apps Script, dan melakukan `git commit` & `git push` untuk menyinkronkan perubahan ke repositori GitHub.
